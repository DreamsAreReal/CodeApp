# RS-07 — Арт-дирекшн (референсы из Pinterest, деконструкция «своими глазами»)

Причина: пользователь отверг сгенерённые haiku-макеты как «нейрослоп, нет вау». Планка поднята:
продукт должен быть «мега красивым» и «напоминать кино». Референсы взяты из живого Pinterest
(вошёл через расширение), 78 реальных изображений скачаны в `docs/research/refs/{constellation,fui,cinematic}/`,
контактные листы — `docs/research/refs/boards/board-1..3.jpg`. Смотрено глазами (не из памяти модели).

## Ключевой урок (анти-slop)
Кино-качество и «вау» берутся из деконструкции реальных работ художников, НЕ из дефолтной эстетики
модели. «Кино» ≠ загромождённый HUD. Кино = драматический СВЕТ (один источник), ОДИН фокус-объект
на экран, глубокие градуированные чёрные (не чистый #000, а сине/фиолетовый near-black), плёночное
зерно + мягкий bloom, сдержанная элегантная типографика, акцентное свечение как источник света —
и при этом ЧИТАЕМОСТЬ. Сдержанность, а не «всё сразу». См. [[design-taste-cinematic-no-aislop]].

## Направление 1 — KNOWLEDGE CONSTELLATION (board-1)
Референсы: Path of Exile / Lost Ark passive-деревья, «Gate of Fates» каменный диск-созвездие,
ability-tree «LIGHT», constellation star-maps, tree-of-knowledge, radial skill-dial.
- Палитра: near-black / тёмно-синий фон, узлы светятся cyan + amber, тонкие связующие линии.
- Свет: узлы — источники света на чёрном; центральный «герой»-узел ярче остальных (глубина).
- Композиция: граф/созвездие как НАВИГАЦИЯ. Прямо ложится на концепт-DAG (пререквизиты) —
  весь curriculum = тёмная звёздная карта, которую зажигаешь по мере освоения.
- Роль в продукте: ГЛАВНЫЙ «вау»-экран (home/навигация). Signature.

## Направление 2 — TECH-CINEMA / FUI (board-2)
Референсы: голографические HUD, DNA-helix data-viz (cyan на чёрном), exploded reactor-схема,
radar/scanner, «SUBJECT ID» диагностика (скан глаза + мозг-сеть + monospace + EEG-волна), «PURSUIT»
красный HUD, loading-элементы. Кино: Blade Runner / Minority Report / Oblivion / Mr. Robot.
- Палитра: чёрный + cyan/teal свечение, редкий red/amber для тревоги/акцента, monospace-текст.
- Свет/motion: сканлайны, лёгкий glitch, пошаговое «раскрытие», плёночное зерно.
- РИСК: легко скатиться в загромождённый Envato-HUD (генерично). Держать РЕСТРЕЙН: одна
  визуализация в фокусе, управляемая пользователем (step-through), не 40 виджетов сразу.
- Роль в продукте: обучающие ВИЗУАЛИЗАЦИИ (hash-таблицы, сортировки, память стек/куча, async-таймлайн,
  attention трансформера) как «кадр анализа» из фильма. Код и алгоритмы как film-tech.

## Направление 3 — CINEMATIC APP SHELL (board-3)
Референсы: тёмные media/gallery-приложения, «Martin Scorsese / Films» editorial, «UNIQUE FILMING
TECHNIQUE» / «RECLARIUM PRODUCTION» плакатная типографика, NOCTURNE «deep focus mode» (тёмные
ambient-волны, стеклянная карточка), «what I finally chose this year» плёночное фото.
- Палитра: премиальный тёмный, филмичные обложки тем, точечный акцент (orange/red/violet), воздух.
- Типографика: крупная элегантная (рафинированный serif для дисплея + гротеск/mono для тела).
- Роль в продукте: реальный SHELL (чтение, профиль, focus-mode, обложки тем) — красиво И юзабельно,
  держит киношность там, где HUD был бы перебором.

## Рекомендованный синтез (моя ставка — на ратификацию пользователем)
Хребет «Обсерватория / Codex киновселенной знаний»: градуированные плёночные чёрные, драматический
свет, один фокус-объект, элегантная типографика, зерно+bloom, cyan/amber свечение как свет.
- HOME/навигация = Направление 1 (Knowledge Constellation) — signature-экран.
- Обучающие визуализации = Направление 2 (FUI), но РЕСТРЕЙН — один фокус, step-through.
- Shell/чтение/focus = Направление 3 (кино-shell).
Инвариант вкуса: «кино = драматическая сдержанность + один герой-фокус + плёночный свет + читаемость»,
НЕ «тёмно и загромождено». Дискретные токены (палитра, шрифты, зерно, свечение) заморозить в ТЗ.

## ВЫБОР ПОЛЬЗОВАТЕЛЯ (2026-07-08): Направление 3 — CINEMATIC APP SHELL
«в сторону 3 работаем. апп шел». Соул = премиальный тёмный shell, editorial-типографика,
focus-mode (NOCTURNE), плёночный грейд, сдержанная элегантность. HUD/FUI (напр-е 2) — только
ТОЧЕЧНО в обучающих визуализациях (step-through алгоритма), НЕ как основной язык. Constellation
(напр-е 1) — можно как один вау-экран навигации, но подчинён shell-эстетике, не диктует всё.
Осталось заморозить на гейте: акцент (cyan/amber/violet/контраст), допустимость грейна, только
тёмная тема (по умолчанию да). Токены вкуса → в brief.md (паспорт вкуса).
