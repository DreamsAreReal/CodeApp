Упаковка значения в object

int a = 5;
object b = a;   // boxing
a = 10;
Console.WriteLine(b);
Вывод 5

b — это упакованная КОПИЯ значения, снятая в момент присваивания и независимая от a. Изменение a на 10 бокс не трогает.
