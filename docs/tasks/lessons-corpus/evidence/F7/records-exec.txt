## c1 / SIGNATURE PANEL — record value equality vs class reference equality
$ POST run-csharp {record Point(int X,int Y); class CPoint; r1==r2 (record), c1==c2 (class), same fields}
stdout: "True False\n"   -> expect "True False"   (record value equality True; class reference equality False)

## c2 — with = nondestructive mutation (new instance, original untouched); synthesized ToString
$ POST run-csharp {var p1=new Point(1,2); var p2=p1 with { X=99 }; WriteLine($"{p1} {p2}")}
stdout: "Point { X = 1, Y = 2 } Point { X = 99, Y = 2 }\n"   -> expect same   (p1 unchanged, p2 is the copy)

## c3 — two DIFFERENT record types, same field -> NOT equal (EqualityContract / runtime type)

stdout: "False\n"   -> expect "False"
