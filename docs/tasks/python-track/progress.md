**Решения.**

  LOCKED, не трогаю) — канон волны: НАТУРАЛЬНАЯ ширина ≤ avail−6.
- Явный `w:` у узла — легальная страховка (size-hint из ладдера, не x/y escape-hatch;
  WIDTH-ON-LADDER соблюдён).
- Рентген-зона: generic `.vz-zone.xray` (грефт B2 турнира) — dashed-контур для
  runtime-невидимого (__closure__ в F4; переиспользовать для gi_frame в F6).

**Грабли (правило на все будущие уроки).**
- ПЕРЕД коммитом урока гонять `node verify/_fit-margins.mjs`: пусто = ок; лейбл с
  натуральным запасом <6px — компактить текст или поднимать w на ступень. Linux
  (FreeType) шире мака до ~4px — margin ≥6 обязателен.
- Опасные длины ascii-чипов: 8–9 симв (бокс 72) и 11–16 симв (бокс 96/120) — natural
  вылезает за avail; безопасно ≤7 или ≤10 симв, либо явный w на ступень выше.
- Ширина чипа не должна меняться между сценами в одном ряду (MID-OVL): лейблы
  константные, состояние — через accent/ghost.
- Каскадные replace при патчах: короткий паттерн — подстрока уже заменённого длинного →
  дубль свойства (TS1117); заменять уникальными якорями.
