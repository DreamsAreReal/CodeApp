- **DIS-ARG-ABBREV (конвенция волны, действует на все dis-панели PY-уроков):**
  аргумент опкода, который в сыром выводе `python3.12 -m dis` показан code-объектом

  и бессмыслен, и запрещён чеклистом RS-03 (п.4: без адресов/id()). Любая новая
  dis-панель обязана следовать этой же схеме и держать лог сырого прогона.

  (гонять verify/_fit-margins.mjs перед коммитом урока); долю 2-сценных чип-сегментов
