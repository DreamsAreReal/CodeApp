# HISTORY — что это, где что лежит и почему так

> Компактная карта проекта. Живые доки — только те, что нужны для работы; вся история
> разработки (ТЗ, ресерчи, вердикты оценщиков, скриншоты-доказательства) удалена из
> рабочего дерева и целиком доступна в git-истории (до коммита `4054379` включительно).

## Что это
Telegram Mini App — ежедневный тренажёр глубоких уроков программирования: анимированные
разборы механизмов («живая диаграмма» на SVG) + карточки активного припоминания на
настоящем расписании FSRS-6. Треки: «Фундамент C#» (6 уроков) и «Python для AQA»
(13 уроков — механизмы языка + шпаргалки, под подготовку к собеседованиям).

## Ключевые решения (почему так)
- **Уроки = данные** (`app/src/lessons/*.ts` + seed-JSON на бэке): добавить урок = один
  data-файл + строка в реестре; движок и дизайн общие. Размещение узлов — только
  `at:{zone,row,col}` (кривой кадр собрать нельзя — гарантирует `verify/viz-fit.mjs`).
- **Треки не пересекаются**: переключатель на главной, `TRACK_GROUPS` в
  `app/src/lessons/index.ts`; контент Python не ссылается на C#.
- **Точность — риск №1**: каждое утверждение с первоисточником (docs.python.org / PEP /
  MS Learn), цитаты дословные; эталоны карточек = реальный stdout (C# — авторский
  `run-csharp`, Python — локальный python3.12, версия пиннится).
- **Оценку свободных ответов LLM-судьёй не используем** — только исполняемая сверка
  (typed stdout) или структурное само-объяснение.
- **Деплой**: push в main → тесты (гейт) → reusable workflow `deploy-hub` (образ в GHCR
  по sha, VPS-раннер, health-гейт, авто-откат). Вызывающему джобу нужен
  `permissions: {packages: write}`.
- **FSRS-6** — верный порт py-fsrs 6.3.1, фаззинг выключен (детерминизм для тестов).

## Известные хвосты (кандидаты волны 2)
- Дозатора новых карточек нет: свежая очередь отдаёт все new разом (сессия «59 карточек»).
- Собес-тренажёр (блоки 10–11 конспекта): открытые вопросы + STAR-кейсы — выбран
  пользователем следующим.
- Бэк: валидация itemId в /api/review (фантомные review), 400 вместо 500 на кривой grade,
  XP-ферма при переответе пройденных карточек.
- dis-кадры в уроки M6/M7/M9 (signature-ось плотнее).
